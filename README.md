# Data-Science-Intern-project-Primetrade.ai
Trader Performance vs Market Sentiment

Data Science Intern Assignment – Primetrade.ai

Objective

This project analyzes the relationship between Bitcoin market sentiment (Fear/Greed Index) and trader behavior on Hyperliquid.
The goal was to understand how sentiment regimes influence trading activity, profitability, and risk-taking behavior, and to derive actionable strategy insights based on the findings.

Dataset:
Two datasets were used:
Bitcoin Fear/Greed Index
Hyperliquid historical trader data

Both datasets were aligned at the daily level using normalized timestamps.

Interactive Dashboard

A lightweight Streamlit dashboard is included to explore the results interactively.

To run locally:

pip install -r requirements.txt
streamlit run app.py
